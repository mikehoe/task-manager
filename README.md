# Python Task Manager

A simple console-based task management application that will gradually evolve into a complete backend application 
with database support, a FastAPI REST API, and AI-powered features.

## Features

- Add a task
- List tasks
- Mark a task as completed
- Delete a task
- Exit the application

## Task Model

Each task contains:
- an identifier,
- a title,
- a completion status.

## Roadmap

The application will be extended with:
- File storage
- Unit tests
- SQLite
- PostgreSQL
- FastAPI
- Docker
- AI integration
